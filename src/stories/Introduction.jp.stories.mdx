# Portfoliem design system

## Language

- [English](?path=/story/introduction-en--page)
- 日本語

## 🔥 Mission

この個人開発のミッションは”ソフトウェアを通じて複雑な問題をシンプルに、美しいビジュアルにデザインしなおす”ことです。

世界中にインターネットが広まり、脳神経のようにあらゆる情報が交差するようになりました。これにより、人類は高度な情報処理能力を手に入れることができましたが、それと同時に複雑化した情報に対応しなければならなくなりました。そこで、私はこのような複雑な問題をソフトウェアとデザインの力で解決したいと考え、ソフトウェアや Web アプリケーションをつくっていきたいと考えています。

## 👀 Vision

アプリの UI コンポーネントをビジネスロジック、データ、コンテキストから分離し、開発を民主化したいと考えています。再利用可能なコンポーネントとガイドラインは貢献と開発に役に立ちます。Portfoliem design system はすべてのコミュニティメンバーができるだけ簡単に貢献できるようにします。

## 👏 貢献方法

バグや変更が必要だと気がついたら GitHub にある[contribution guide](https://github.com/chidoriashi1990/portfoliem-storybook/blob/master/CONTRIBUTING.md)に従って、私たちに伝えてください。プロセス全体を通してあなたを巻き込み、あなたのニーズを満たすよう努めます。

[GitHub](https://github.com/chidoriashi1990/portfoliem-storybook)でデザインと改善のアイデアについて話し合うのを楽しみにしています。

## 💎 Values

### Go Challenging

ただ実装するのではなく新しい挑戦にも同時に取り組みます。

### Be a Beautiful

既存のものより美しく、価値ある体験を提供します。

### To be Consistent

データやデザインに一貫性を持たせ、誰もが同じように認識できるようにします。

## ✨ Brand Identity

### 一貫性のあるデザイン

Material Design をコアコンポーネントに据え置き、ロゴ、カラーパレット、タイポグラフィなどの構成要素によって一貫性を保ちます。

### 遊び心と表現力

ユーザーの感情に訴えかけるような柔軟で、創造的な表現を取り入れます。イラスト、アニメーション、写真へのアプローチにより新たな表現力を探究して行きます。

### 洗練されたアイコン

一目見ただけでわかり、いかなるシーンでも存在感を示すアイコンでサービスを表現します。
